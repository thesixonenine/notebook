## 常用系统工作命令

### echo 命令

格式: 

```
echo [字符串|$变量]
```

用途: 用于在终端输出字符串或变量提取后的值

示例: 

```SHELL
# 打印字符 hello shell
echo "hello shell"
# 打印变量SHELL的值
echo $SHELL
```

### date 命令

格式: 

```
date [选项][+指定的格式]
```

用途: 显示和设置系统的时间或日期

示例: 

```SHELL
# 输出当前系统时间
date
# 格式化输出当前系统时间
date "+%Y-%m-%d %H:%M:%S"
# 设置当前系统时间
date -s 2008/08/08
date -s 08:08:08
date -s "08:08:08 2008-08-08"
date -s "08:08:08 20080808"
date -s "2008-08-08 08:08:08"
date -s "20080808 08:08:08"
# 查看今天是今年中的第几天
date "+%j"
```
