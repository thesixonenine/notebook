#### 算术运算符

select子句中使用运算符

```SQL
SELECT "测试" AS "常数", `id`, `name` AS "商品名称", price * 3 AS "价格乘以三" FROM product;
```

```SQL
SELECT 3 * 9 AS "三乘以九的结果";
```

- 加法运算: +
- 减法运算: - 
- 乘法运算: *
- 除法运算: /

null参与运算的时候, 值均为null, 即使是 null / 0

```SQL
SELECT 3 * NULL AS "三乘以NULL的结果";
```

Oracle数据库不允许省略from子句
